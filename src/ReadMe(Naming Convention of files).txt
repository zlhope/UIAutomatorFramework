Naming Convention

1. Script Name -> typeOftest underscore module underscore (optional support device required denoted as MS)  
 	e.g. Sanity_All.py,Sanity_All_MS.py, Stability_Telephony_MS

2. Config file-> Model underscore android version underscore Config 
	e.g. TC700H_KK_Config, TC75_KK_Config

3. ModuleFunctionsFile -> Model underscore ModuleFunctions
	e.g. TC700H_KK_ModuleFunctions, TC75_KK_ModuleFunctions
